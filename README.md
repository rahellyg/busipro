# ביזנס פרו - אתר בניית אתרים

אתר מקצועי לבניית אתרים עם דף נחיתה ובלוג בעברית.

## התקנה והרצה

### דרישות מוקדמות
- Node.js (גרסה 16 ומעלה)
- npm או yarn

### התקנת התלויות
```bash
npm install
```

### הרצת השרת לפיתוח
```bash
npm run dev
```

האתר יפתח בכתובת: `http://localhost:5173`

### בניית גרסת ייצור
```bash
npm run build
```

### תצוגה מקדימה של גרסת ייצור
```bash
npm run preview
```

## מבנה הפרויקט

```
buispro/
├── src/
│   ├── components/      # קומפוננטות משותפות
│   │   ├── Navbar.jsx
│   │   └── Footer.jsx
│   ├── pages/           # דפי האתר
│   │   ├── Home.jsx      # דף נחיתה
│   │   ├── Blog.jsx      # רשימת פוסטים
│   │   └── BlogPost.jsx  # פוסט בודד
│   ├── data/             # נתונים
│   │   └── blogPosts.js  # פוסטי הבלוג
│   ├── App.jsx           # קומפוננטת ראשית
│   └── main.jsx          # נקודת כניסה
├── index.html
└── package.json
```

## התאמה אישית

### עריכת תוכן הבלוג
ערוך את הקובץ `src/data/blogPosts.js` כדי להוסיף או לערוך פוסטים.

### שינוי צבעים
ערוך את המשתנים ב-`src/index.css`:
```css
:root {
  --primary-color: #2563eb;
  --secondary-color: #10b981;
  /* ... */
}
```

### שינוי תוכן דף הנחיתה
ערוך את הקובץ `src/pages/Home.jsx` כדי לשנות את התוכן.
