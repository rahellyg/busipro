# הוראות התקנה מהירות - שימוש ב-ChatGPT

## מה צריך לעשות?

### שלב 1: קבלת API Key מ-OpenAI

1. היכנס ל: https://platform.openai.com/api-keys
2. היכנס לחשבון שלך (או צור חשבון חדש)
3. לחץ על "Create new secret key"
4. העתק את ה-API key (הוא יופיע רק פעם אחת!)

### שלב 2: יצירת קובץ .env

1. בתיקיית הפרויקט (buispro), צור קובץ חדש בשם `.env`
2. הוסף את השורות הבאות:

```env
VITE_OPENAI_API_KEY=sk-your-api-key-here
VITE_AI_PROVIDER=openai
```

**חשוב:** החלף את `sk-your-api-key-here` ב-API key האמיתי שקיבלת!

### שלב 3: הפעלה מחדש

1. עצור את שרת הפיתוח (אם הוא רץ) - לחץ `Ctrl+C` בטרמינל
2. הפעל מחדש: `npm run dev`

### שלב 4: בדיקה

1. מלא את הטופס "בונה AI"
2. לחץ על "צור תצוגה מקדימה"
3. המערכת תשלח בקשה ל-ChatGPT ותציג:
   - תמונות שנוצרו על ידי DALL-E (קטלוג, עגלה, מבצעים)
   - אתר אינטראקטיבי עם תוכן מותאם

## מה קורה אם אין API key?

אם אין API key, המערכת תעבוד עם נתונים בסיסיים (ללא AI):
- לא תהיינה תמונות מ-DALL-E
- התוכן יהיה בסיסי (לא מותאם מ-AI)
- האתר עדיין יעבוד, אבל פחות מותאם

## עלויות

**ChatGPT (GPT-4o-mini):**
- ~$0.15 לכל 1M tokens
- כל אתר = ~2,000-5,000 tokens
- **עלות לאתר**: ~$0.001-0.0015

**DALL-E 3 (תמונות):**
- ~$0.04 לתמונה
- כל אתר = 3 תמונות
- **עלות לאתר**: ~$0.12

**סה"כ לאתר**: ~$0.12-0.13

## פתרון בעיות

### שגיאה: "API Key לא נמצא"
- ודא שיצרת קובץ `.env` בתיקיית הפרויקט
- ודא שהשם נכון: `VITE_OPENAI_API_KEY`
- ודא שהערך מתחיל ב-`sk-`
- הפעל מחדש את שרת הפיתוח

### שגיאה: "שגיאה ב-OpenAI API"
- ודא שה-API key תקף
- בדוק שיש לך יתרה/קרדיט בחשבון OpenAI
- פתח את ה-console בדפדפן (F12) כדי לראות שגיאות מפורטות

### תמונות לא מוצגות
- ודא שיש לך קרדיט ב-OpenAI
- DALL-E 3 דורש תשלום (לא חינמי)
- בדוק את ה-console לשגיאות

## אלטרנטיבות (אם אין OpenAI)

### Google Gemini (חינמי)
```env
VITE_GEMINI_API_KEY=your-key-here
VITE_AI_PROVIDER=gemini
```

### Anthropic Claude
```env
VITE_ANTHROPIC_API_KEY=sk-ant-your-key-here
VITE_AI_PROVIDER=claude
```

**הערה:** רק OpenAI תומך ב-DALL-E ליצירת תמונות. Gemini ו-Claude יוצרים רק תוכן טקסט.
